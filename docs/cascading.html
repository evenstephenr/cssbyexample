CSS by Example: cascading

In the previous example, three different stylesheets were being referenced on the page.

<html>   <head>     <!-- internal link to stylesheet -->     <link rel="stylesheet" href="./styles.css" />     <!-- external link to stylesheet -->     <link rel="stylesheet" href="https://external.domain/styles.css" />     <style>       /* inline style tag */     </style>   </head>   <body>     <!-- -->   </body> </html>

All three stylesheets will apply to the page, and some of them may override or extend others.

This is known as cascading behavior.

overrides using color, extending using %, etc.

you can break cascading behavior using specificity (provide link), but proceed with caution
