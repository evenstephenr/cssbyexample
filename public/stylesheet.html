CSS by Example: stylesheet

A CSS stylesheet is a set of CSS rules that will apply to the current document.

CSS rules are composed of selector groups, their nested properties, and their property values.

Here's an example of a basic stylesheet

html, body {   font-family: Arial, Helvetica, sans-serif;   margin: 0;   padding: 0;   background-color: #f3f3f3; } body {   width: 90%;   max-width: 450px;   background-color: #fff; }

CSS files have the extension .css.

You can import a CSS file in a html document using the <link> tag, or declare styles inline using the <style> tag in a HTML <head>.

<html>   <head>     <!-- internal link to stylesheet -->     <link rel="stylesheet" href="./styles.css" />     <!-- external link to stylesheet -->     <link rel="stylesheet" href="https://external.domain/styles.css" />     <style>       /* inline style tag */     </style>   </head>   <body>     <!-- -->   </body> </html>

Either method will apply your stylesheet to your document.
